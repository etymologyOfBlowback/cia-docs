* What These Tools Do

** Redaction Identification

1. **Extracts redaction box coordinates** from PDF structure
   - Uses pdfplumber to read PDF vector objects
   - Identifies BLACK RECTANGLES in the PDF
   - Gets exact pixel coordinates

2. **Creates annotated PNG files**
   - Extracts page images from PDF
   - Draws red boxes marking redaction locations
   - Saves one PNG per redacted page

3. **Generates CSV reports**
   - Documents each redaction box
   - Records pixel coordinates
   - Calculates redaction statistics
   - Tracks percentage of page redacted

** The Key Insight

Redactions are stored as VECTOR OVERLAY BOXES in the PDF structure, separate from the image data. By extracting these boxes, we can precisely identify what was meant to be hidden.

* Example: iran-cia-main.7.pdf

Running the tool on this document produced:

**Results:**
- 14 pages with redactions (out of 22 total)
- 45 total redaction boxes
- Largest: Page 11 with 39.2% of page redacted
- Smallest: 0.2% of page

**Output:**
- 14 annotated PNG files (one per redacted page)
- 1 CSV report with 45 rows (one per redaction box)

* Quick Start

** Single PDF

#+BEGIN_SRC bash
python3 identify_redactions.py iran-cia-main.7.pdf
#+END_SRC

**Output:**
- =iran-cia-main.7_page_N_redactions_marked.png= (annotated images)
- =iran-cia-main.7_redaction_report.csv= (statistics)

** Batch: All 17 PDFs

#+BEGIN_SRC bash
python3 batch_identify_redactions.py
#+END_SRC

**Output:**
- One annotated PNG set per PDF
- One CSV report per PDF
- Combined summary (optional)

* CSV Report Format

The generated CSV files contain:

#+BEGIN_EXAMPLE
PDF File,Page Number,Redaction Box #,X0,Y0,X1,Y1,Width,Height,Area,Percent,Total Boxes,Total %,Image Size,Time
iran-cia-main.7.pdf,1,1,187,532,270,554,83,22,1826,0.4%,2,0.7%,590x843,2026-02-20T15:19:33
iran-cia-main.7.pdf,4,1,212,316,365,340,153,24,3672,0.7%,3,4.0%,590x843,2026-02-20T15:19:33
iran-cia-main.7.pdf,11,1,94,122,542,554,448,432,193536,38.9%,2,39.2%,590x843,2026-02-20T15:19:33
#+END_EXAMPLE

**Key columns:**
- X0, Y0, X1, Y1 = Pixel coordinates of redaction box
- Area = Size in pixels²
- Percent = What % of page this box covers
- Total % = Total % of page with all redactions
