* Overview

Tools for identifying, marking, and documenting redacted regions in PDF documents. These scripts analyze the PDF structure to locate redaction boxes and create annotated PNG files with CSV reports.

* Files

| Script | Purpose |
|---|---|
| =identify_redactions.py= | Identify redactions in single or multiple PDFs |
| =batch_identify_redactions.py= | Batch process all iran-cia-*.pdf files |

* How It Works

** Process Flow

1. **Extract redaction coordinates** from PDF structure using pdfplumber
2. **Identify black rectangular boxes** in the PDF's vector objects
3. **Extract page images** and convert to PNG
4. **Mark redacted regions** with red boxes and coordinates
5. **Generate CSV report** with detailed redaction statistics

** Key Insight

The script identifies redactions by looking for BLACK RECTANGLES in the PDF's vector graphics layer. These are the overlay boxes that sit on top of the document image.

* Interpretation Guide

** Reading the Annotated PNGs

When viewing the marked PNG files:

- **Red boxes** = Regions that were redacted in the original PDF
- **Box size** = How much content was hidden
- **Number of boxes** = How many separate redactions on that page
- **Unredacted areas** = Content that was publicly visible

**Important:** The red boxes show where the redactions WERE, not what was hidden. The underlying content is now visible as regular image content.

** Reading the CSV Reports

Use the CSV to:

1. **Find heavily redacted pages** - Sort by "Total Page Redaction %" column
2. **Identify specific boxes** - Use X0, Y0, X1, Y1 coordinates
3. **Analyze patterns** - Which pages/documents have most redactions
4. **Track coverage** - "Area" column shows total hidden pixels

#+END_SRC

* Troubleshooting

** "No redaction boxes detected"

This means the PDF doesn't have any black rectangle objects in the vector layer.

**Possible reasons:**
- PDF was created with embedded image redactions (not vector overlays)
- Redactions use different color than black
- Document has no redactions

**Solution:** Check if redactions are visible in original PDF

** CSV file missing data

Some columns may be empty if:
- Image size varies between pages
- Redactions have unusual dimensions

**Solution:** These are still valid entries; null values are acceptable

** Annotated PNG looks wrong

If the red boxes don't align with expected locations:

- PDF coordinates may have been transformed
- Image scaling might be different
- Check image size matches

**Solution:** Compare with original PDF page side-by-side
